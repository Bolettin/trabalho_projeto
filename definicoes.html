Definições


Servidor FTP

Um servidor FTP é um computador que permite o envio e recebimento de arquivos através do protocolo FTP (File Transfer Protocol). Ele pode ser um computador dedicado ou uma máquina virtual. O FTP é um protocolo de comunicação que permite a transferência de arquivos entre um servidor e um cliente. Ele é um dos protocolos mais antigos e simples ainda em uso para a transferência de dados.

JavaScript

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.

Docker

O Docker é uma plataforma open source que facilita a criação e administração de ambientes isolados. Ele possibilita o empacotamento de uma aplicação ou ambiente dentro de um container, se tornando portátil para qualquer outro host que contenha o Docker instalado.
